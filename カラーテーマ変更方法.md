# カラーテーマ変更方法

## 概要

アプリ全体のカラーを簡単に変更できるように、CSS変数（カスタムプロパティ）を使用しています。

## 変更方法

`src/styles/variables.css` ファイルを編集することで、アプリ全体の色を変更できます。

## カラー変数の一覧

### 背景グラデーション
```css
--bg-gradient-start: #667eea;  /* 背景グラデーションの開始色 */
--bg-gradient-end: #764ba2;    /* 背景グラデーションの終了色 */
```

### カード背景色
```css
--card-bg: rgba(250, 248, 255, 0.95);  /* カードの背景色（透明度付き） */
--card-gradient-start: #faf8ff;        /* カードグラデーションの開始色 */
--card-gradient-end: #f0edf5;          /* カードグラデーションの終了色 */
```

### ホバー時のグラデーション
```css
--hover-gradient-start: #667eea;  /* ホバー時のグラデーション開始色 */
--hover-gradient-end: #764ba2;    /* ホバー時のグラデーション終了色 */
```

### テキスト色
```css
--text-primary: #333;    /* メインテキストの色 */
--text-secondary: #666;  /* サブテキストの色 */
```

### ボーダー・シャドウ
```css
--border-color: #ccc;                           /* ボーダーの色 */
--shadow-color: rgba(0, 0, 0, 0.1);            /* 通常の影の色 */
--shadow-hover-color: rgba(102, 126, 234, 0.4); /* ホバー時の影の色 */
--button-shadow: rgba(102, 126, 234, 0.3);     /* ボタンの影の色 */
--button-shadow-hover: rgba(102, 126, 234, 0.4); /* ボタンホバー時の影の色 */
```

## 使用例

### 青系のテーマに変更する場合
```css
:root {
  --bg-gradient-start: #4facfe;
  --bg-gradient-end: #00f2fe;
  --card-bg: rgba(240, 248, 255, 0.95);
  --card-gradient-start: #f0f8ff;
  --card-gradient-end: #e6f3ff;
  --hover-gradient-start: #4facfe;
  --hover-gradient-end: #00f2fe;
}
```

### 緑系のテーマに変更する場合
```css
:root {
  --bg-gradient-start: #43e97b;
  --bg-gradient-end: #38f9d7;
  --card-bg: rgba(240, 255, 248, 0.95);
  --card-gradient-start: #f0fff4;
  --card-gradient-end: #e6f7ed;
  --hover-gradient-start: #43e97b;
  --hover-gradient-end: #38f9d7;
}
```

### オレンジ系のテーマに変更する場合
```css
:root {
  --bg-gradient-start: #fa709a;
  --bg-gradient-end: #fee140;
  --card-bg: rgba(255, 248, 240, 0.95);
  --card-gradient-start: #fff8f0;
  --card-gradient-end: #fff0e6;
  --hover-gradient-start: #fa709a;
  --hover-gradient-end: #fee140;
}
```

## 注意事項

- 色を変更した後は、ブラウザをリロードしてください
- 透明度（rgba）を使用している場合は、背景との調和に注意してください
- コントラスト比を確保して、アクセシビリティに配慮してください
