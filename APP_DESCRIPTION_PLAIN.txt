ご当地マンホールアプリ - 詳細説明

アプリ概要

「ご当地マンホールアプリ」は、日本各地のご当地マンホールを撮影・管理するためのPWA（Progressive Web App）アプリケーションです。オフライン環境でも動作し、スマートフォンやタブレットのホーム画面に追加することで、ネイティブアプリのように使用できます。

主な機能

基本機能（無料）

写真管理機能
- カメラから直接撮影、または端末内の画像ファイルから写真を追加可能
- 都道府県別に自動整理され、撮影枚数を一覧表示
- EXIF情報から撮影日時を自動取得
- 画像の自動圧縮により、ストレージ容量を効率的に使用

サムネイル調整機能
- 写真ごとにサムネイル表示範囲を自由に調整
- ドラッグ&ドロップで位置を変更
- ハンドル操作でサイズを独立して調整（縦横比固定なし）
- 一覧画面と詳細画面の両方から編集可能

メモ機能
- 各写真にメモを追加・編集可能
- 撮影場所やマンホールの特徴を記録
- 検索機能と連携してメモ内容で検索

データ管理機能
- 全データをJSON形式でエクスポート
- 別デバイスへのデータ移行に対応
- モバイル端末ではShare APIを利用した簡単な共有
- ストレージ使用量の確認機能
- 画像圧縮品質の調整（高品質・標準・低容量から選択）

有料機能

お気に入り機能
- お気に入りの写真を⭐マークで管理
- 写真一覧画面からワンタップで追加/削除
- 専用のお気に入り一覧ページから素早くアクセス
- 検索結果でもお気に入りフィルターを適用可能

検索機能
- メモ内容での全文検索
- 都道府県名での検索
- お気に入りフィルターと組み合わせた検索
- リアルタイムでの検索結果表示

達成率表示機能
- 都道府県ごとの撮影進捗を視覚的に表示
- 全マンホール数の設定と管理
- 達成率のパーセンテージ表示
- 100%達成時にコンプマーク（🎉）を自動表示
- 最終更新日時の記録

詳細な統計情報
- 地方別（北海道・東北、関東、中部、近畿、中国・四国、九州・沖縄）での一覧表示
- 各都道府県の撮影枚数を常に表示
- 非表示機能で管理しやすい都道府県を一時的に非表示に設定可能

技術的特徴

PWA対応
- Service Workerによるオフライン動作
- ホーム画面への追加でアプリのように使用可能
- 自動更新機能で最新バージョンに自動アップデート

高速なデータアクセス
- IndexedDBを使用したクライアント側データベース
- 大量の写真データも高速に表示
- オフライン環境でも完全に動作

画像最適化
- Canvas APIを使用した画像処理
- 自動リサイズと圧縮によりストレージを効率的に使用
- サムネイルと元画像を分けて保存

レスポンシブデザイン
- スマートフォン・タブレット・PCすべてに対応
- タッチ操作に最適化されたUI
- 画面サイズに応じて自動レイアウト調整

使い方

初回セットアップ
1. ブラウザでアプリにアクセス
2. アドレスバーのインストールアイコンをタップ
3. 「ホーム画面に追加」を選択
4. ホーム画面からアプリを起動

写真の追加
1. 都道府県一覧から都道府県を選択
2. 「写真を追加」ボタンをタップ
3. カメラで撮影するか、ファイルから選択
4. メモを入力（任意）
5. 保存をタップ

サムネイルの調整
1. 写真詳細画面を開く
2. 「サムネイル範囲を調整」ボタンをタップ
3. 画像上の選択範囲をドラッグで移動
4. 青いハンドルをドラッグしてサイズを調整
5. 「サムネイルを保存」をタップ

データのバックアップ
1. 設定画面を開く
2. 「データをエクスポート」をタップ
3. ファイルがダウンロードされる（モバイルでは共有）
4. 別デバイスで「データをインポート」からファイルを選択

データの保存場所

すべてのデータはブラウザのIndexedDBに保存されます。サーバーへの送信は一切行わないため、プライバシーが保護され、完全にオフラインで動作します。データは端末ローカルにのみ保存されるため、ブラウザのデータをクリアすると消去される可能性があります。定期的なエクスポートを推奨します。

対応環境

- Chrome / Edge（推奨）
- Firefox
- Safari（iOS 11.3以上）
- その他のPWA対応ブラウザ

モバイル端末での使用を推奨します。特にスマートフォンでの撮影・管理が快適です。
