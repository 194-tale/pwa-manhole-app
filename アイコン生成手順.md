# PWAアイコン生成手順

## 方法1: ブラウザツールを使用（推奨）

1. **アイコン生成ツールを開く**
   - プロジェクトルートの `generate-icons.html` をブラウザで開きます
   - ファイルをダブルクリックするか、ブラウザにドラッグ&ドロップしてください

2. **アイコンの設定**
   - 「アイコンのテキスト」: デフォルトは「マンホール」（変更可能）
   - 「背景色」: デフォルトは紫 (#8b5cf6)（変更可能）
   - 「テキスト色」: デフォルトは白 (#ffffff)（変更可能）

3. **アイコンを生成**
   - 「🎨 アイコンを生成」ボタンをクリック
   - 3つのサイズのアイコンのプレビューが表示されます

4. **アイコンをダウンロード**
   - 各アイコンの「💾 ダウンロード」リンクをクリック
   - 以下のファイルがダウンロードされます：
     - `icon-192x192.png`
     - `icon-512x512.png`
     - `apple-touch-icon.png`

5. **アイコンを配置**
   - ダウンロードした3つのファイルを `public/icons/` ディレクトリにコピーしてください

## 方法2: Node.jsスクリプトを使用

```bash
# プロジェクトルートで実行
node generate-icons.js
```

注意: パスの問題がある場合は、方法1を使用してください。

## 確認

アイコンを配置後、以下を確認してください：

- [ ] `public/icons/icon-192x192.png` が存在する
- [ ] `public/icons/icon-512x512.png` が存在する
- [ ] `public/icons/apple-touch-icon.png` が存在する

配置後、開発サーバーを再起動して動作確認してください。
