# PWAアイコン

このディレクトリには、PWAアプリのアイコン画像を配置します。

## 必要なアイコン

以下のアイコン画像を配置してください：

- `icon-192x192.png` (192x192px)
- `icon-512x512.png` (512x512px)
- `apple-touch-icon.png` (180x180px) - iOS用

## アイコンの生成方法

### 方法1: ブラウザツールを使用（推奨）

1. プロジェクトルートの `generate-icons.html` をブラウザで開く
2. テキストと色を設定
3. 「アイコンを生成」ボタンをクリック
4. 各アイコンの「ダウンロード」リンクをクリックして保存
5. ダウンロードしたファイルを `public/icons/` ディレクトリに配置

### 方法2: Node.jsスクリプトを使用

```bash
# sharpパッケージをインストール
npm install --save-dev sharp

# アイコンを生成
node generate-icons.js
```

## アイコンのデザイン

- **テキスト**: 「マンホール」（変更可能）
- **背景色**: 紫 (#8b5cf6)
- **テキスト色**: 白 (#ffffff)

デザインは後でカスタマイズできます。
