# ご当地マンホールアプリ

日本各地のご当地マンホールを撮影・管理するPWA（Progressive Web App）アプリケーション。

## 📱 デモ

**ライブデモ:** [https://194-tale-pwa-manhole-app.vercel.app](https://194-tale-pwa-manhole-app.vercel.app)

## ✨ 主な機能

### 基本機能（無料）

- 📷 **マンホール写真の撮影・保存**
  - カメラまたはファイルから写真を追加
  - 都道府県別に整理・管理
  - 自動的なEXIF日時取得

- ✂️ **サムネイル範囲の調整**
  - ドラッグ&ドロップで位置調整
  - ハンドル操作でサイズ調整
  - 詳細画面からも編集可能

- 📝 **メモ機能**
  - 撮影場所やマンホールの特徴を記録
  - いつでも編集・更新可能

- 💾 **データのエクスポート/インポート**
  - JSON形式で全データをバックアップ
  - 別デバイスへのデータ移行に対応
  - モバイルでは共有機能も利用可能

### 有料機能

- ⭐ **お気に入り機能**
  - お気に入りの写真をマーク
  - お気に入り一覧から素早くアクセス
  - 写真一覧からワンタップで追加/削除

- 🔍 **検索機能**
  - メモ内容で検索
  - 都道府県名で検索
  - お気に入りフィルターも利用可能

- 📊 **達成率表示**
  - 都道府県ごとの撮影進捗を表示
  - 全マンホール数の設定と管理
  - 達成率100%でコンプマーク表示

- 🗺️ **詳細な統計情報**
  - 地方別の進捗確認
  - 最終撮影日時の表示

## 🚀 使い方

### インストール（PWAとして）

1. ブラウザでアプリにアクセス
2. アドレスバーのインストールアイコンをクリック
3. 「ホーム画面に追加」を選択
4. ホーム画面からアプリを起動

### 基本的な使い方

1. **写真を追加**
   - 都道府県を選択
   - 「写真を追加」ボタンをクリック
   - カメラで撮影するか、ファイルを選択

2. **サムネイルを調整**
   - 写真詳細画面で「サムネイル範囲を調整」をクリック
   - ドラッグで位置、ハンドルでサイズを調整
   - 「サムネイルを保存」をクリック

3. **メモを追加・編集**
   - 写真詳細画面で「編集」ボタンをクリック
   - メモを入力して保存

4. **お気に入りに追加（有料機能）**
   - 写真一覧画面の各写真の左上にある☆ボタンをクリック
   - ⭐マークが表示されれば完了

## 🔧 技術スタック

- **フロントエンド**: React 19 + TypeScript
- **ビルドツール**: Vite
- **ルーティング**: React Router v7
- **データベース**: IndexedDB (idb)
- **PWA**: vite-plugin-pwa (Workbox)
- **画像処理**: Canvas API
- **スタイリング**: CSS Modules

## 📦 セットアップ（開発者向け）

### 必要な環境

- Node.js 18以上
- npm または yarn

### インストール

```bash
npm install
```

### 開発サーバーの起動

```bash
npm run dev
```

ブラウザで `http://localhost:5173` にアクセスします。

### ビルド

```bash
npm run build
```

`dist`フォルダにビルド結果が出力されます。

### プレビュー

```bash
npm run preview
```

ビルド結果をローカルでプレビューします。
