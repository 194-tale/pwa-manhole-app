# 開発サーバー起動方法

## 方法1: バッチファイルを使用（推奨）

`start-dev.bat` をダブルクリック

## 方法2: コマンドプロンプトから実行

1. `pwa-manhole-app` フォルダをエクスプローラーで開く
2. アドレスバーに `cmd` と入力してEnter
3. 以下のコマンドを実行：

```cmd
npm run dev
```

## 方法3: PowerShellから実行

1. `pwa-manhole-app` フォルダをエクスプローラーで開く
2. アドレスバーに `powershell` と入力してEnter
3. 以下のコマンドを実行：

```powershell
npm run dev
```

## 起動後のアクセス方法

ブラウザで以下のURLにアクセス：
```
http://localhost:5173
```

## 停止方法

ターミナルで `Ctrl + C` を押す
