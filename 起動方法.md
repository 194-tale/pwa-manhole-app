# 開発サーバー起動方法

## 手動で起動する場合

1. ターミナル（PowerShellまたはコマンドプロンプト）を開く
2. 以下のコマンドでプロジェクトディレクトリに移動：

```powershell
cd "C:\Users\mhero\OneDrive\デスクトップ\cursor\pwa-manhole-app"
```

3. 開発サーバーを起動：

```powershell
npm run dev
```

4. ブラウザで以下のURLにアクセス：

```
http://localhost:5173
```

## 注意事項

- 初回起動時にIndexedDBが自動的に初期化されます
- 47都道府県のデータが自動的に作成されます
- 現在は写真追加機能が未実装のため、写真は表示されません
